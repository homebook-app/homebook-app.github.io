# Getting Started

Homebook ist als zentrale Plattform f\u00fcr private Homeserver gedacht.

## Ziel

- Familien und Einzelpersonen im Alltag unterst\u00fctzen
- Datenhoheit behalten
- Module Schritt f\u00fcr Schritt aktivieren

## Setup-Ansatz

1. Homebook auf deinem Server deployen
2. Basis-Module aktivieren (`finance`, `wiki`, `meal-plan`)
3. Nutzer und Rollen einrichten
4. Optional API-Integrationen anbinden
